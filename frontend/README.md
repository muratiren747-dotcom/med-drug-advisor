# Psy-Med Advisor — Frontend

React-based frontend for the Psy-Med Advisor psychiatric drug interaction analysis tool.

## Getting Started

Install dependencies:
npm install

Start development server:
npm start

Open http://localhost:3000 in your browser.

## Tech Stack
- React
- React Router
- Axios

## Notes
- Requires Flask backend running on port 5000
- See /backend for API documentation
